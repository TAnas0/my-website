Projects

  • Project One: Distributed File System

    A scalable file-sharing platform built with Node.js and AWS S3, allowing users to securely upload, manage, and retrieve large files. This project involves complex backend architectures, including file versioning and data duplication for backup.

    Technologies: Node.js, Express, AWS S3, OAuth2, REST API, MongoDB

  • Project Two: Real-Time Analytics Dashboard

    Designed and developed a real-time analytics dashboard for monitoring social media engagement metrics. The dashboard integrates with APIs from various social platforms to visualize engagement data in real-time.

    Technologies: Python, Flask, PostgreSQL, WebSockets, Chart.js, Docker

  • Project Three: Automated Data Pipeline

    An ETL pipeline built with Apache Airflow that automatically ingests data from multiple third-party APIs, processes it, and loads it into a data warehouse. The pipeline is designed to scale and handle large volumes of data efficiently.

    Technologies: Apache Airflow, Python, PostgreSQL, REST APIs, Docker

  • Project Four: Personal Finance Tracker

    A personal finance web application that allows users to track their expenses and savings over time. The application includes features for categorizing transactions, setting budget goals, and generating financial reports.

    Technologies: Django, Python, SQLite, Bootstrap, JavaScript
